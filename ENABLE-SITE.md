# Your site is not live yet — enable GitHub Pages

Your code is on GitHub, but **GitHub Pages must be turned on** for https://nikulkumar-dabhi.github.io to work.

## Do this (takes about 1 minute)

### 1. Open the Pages settings

**Click this link:**  
**https://github.com/Nikulkumar-Dabhi/nikulkumar-dabhi.github.io/settings/pages**

(If you’re not logged in, log in to GitHub first.)

---

### 2. Under “Build and deployment”

- **Source:** select **“Deploy from a branch”**
- **Branch:** select **“main”**
- **Folder:** select **“/ (root)”**

### 3. Click **Save**

---

### 4. Wait 1–2 minutes

GitHub will build and publish your site. Then open:

**https://nikulkumar-dabhi.github.io**

Your portfolio should be live.

---

## If you don’t see “Pages” in the repo

- Make sure you’re in: **https://github.com/Nikulkumar-Dabhi/nikulkumar-dabhi.github.io**
- Click **Settings** (top menu of the repo)
- In the left sidebar, click **Pages** (under “Code and automation”)
